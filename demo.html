04 SMT Encoding + Solver 2 underdetermined ▾

Formulas are encoded as named constraints in Z3 using assert_and_track — never solver.add. Named tracking enables the solver to return a minimal infeasible core that maps directly back to document locations.

from z3 import * solver = Solver() solver.set("unsat_core", True) # Variables priority_return = Int('priority_return') priority_interest = Int('priority_interest') rank_m1 = Int('rank_m1'); rank_m2 = Int('rank_m2'); rank_b1 = Int('rank_b1') wd_order_b2h = Int('wd_order_b2h') # ... + 14 more variables # Named assertions — assert_and_track throughout solver.assert_and_track( Implies(payment_date, And(priority_return < priority_interest, priority_return < priority_principal)), Bool('A::428b3e2a::1_0_0::C0001')) solver.assert_and_track( And(rank_m1 < rank_m2, rank_m2 < rank_b1), Bool('A::428b3e2a::1_0_0::C0008')) # ... 13 additional named assertions result = solver.check() # → sat

Write-down / write-up structural asymmetry
Write-down allocations are bottom-up (junior tranches absorb losses first). Write-up allocations are top-down (senior tranches recover first). These are inverse orderings — deliberate structural asymmetry, not a contradiction. B-1 Notes absorb losses before M-2 and M-1, but recover after them.

UNDERDETERMINED
EOD interest accrual path
C0011: EOD triggers early redemption. C0012: notes cease to bear interest after early redemption, unless EOD has occurred. Combined: an EOD-triggered redemption leaves notes continuing to accrue interest. The document does not specify whether the Trust has sufficient assets to pay that continuing interest or where it ranks in the waterfall on a post-EOD payment date.

Maturity principal obligation vs Return Amount priority — material gap
C0003 states the Trust "will pay" 100% of outstanding principal at maturity — absolute language. C0009 states Return Amount is paid before any amounts owed on Notes. Under stress conditions where Eligible Investments are insufficient to cover both, these obligations compete. The document does not contain an explicit provision resolving this competition.
