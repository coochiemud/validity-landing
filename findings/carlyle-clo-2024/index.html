01 — The Finding

A coverage-test cascade under constrained proceeds

Carlyle Direct Lending CLO 2024-1 contains a Priority of Payments waterfall. Several clauses require payments to be made through the Debt Payment Sequence when coverage tests fail.

§ 11.1.1.1(F) — Class A/B Coverage Test draw
§ 11.1.1.1(H) — Class C Coverage Test draw
§ 11.1.1.1(K) — Class D Coverage Test draw
§ 11.1.1.1(M) — Interest Diversion Test draw
§ 11.1.2 — Insufficiency of Funds provision

Each draw is conditional. Each draw depends on a coverage test failure. Each draw ultimately competes for available proceeds.

Validity detected that, under a stress state where multiple coverage tests fail at once, the aggregate payment demand can exceed available Interest Proceeds.

The waterfall resolves this by priority: senior claims are paid first, and junior claims receive only what remains.

Validity's contribution is not discovering that junior investors can go short. That is basic structured finance.

Validity's contribution is proving the exact structural condition under which that shortfall occurs.

03 — Why This Matters

Traditional review can describe CLO waterfall risk. Validity can formally verify it.

A lawyer can read the waterfall and explain the priority structure.
A credit analyst can model expected payment behaviour.
A rating agency can assess the deal under stress assumptions.

Validity adds a different capability: it converts the document itself into a logical system and proves whether specified commitments can be jointly satisfied under defined conditions.

That matters because structured finance documents contain thousands of conditional obligations. Many are individually clear. The risk often sits in the interaction between them.

Validity reads those interactions as logic.

04 — What This Is Not

A demonstration, not an allegation

This is not an allegation that Carlyle misdrafted the indenture.

It is not a claim that the CLO is defective.

It is not a claim that rating agencies, lawyers, or investors missed an obvious legal error.

It is a demonstration that formal verification can detect and prove stress-state payment interactions that conventional document review does not express as deterministic logic.

06 — Source Clauses

Five provisions constituting the formal finding

§ 11.1.1.1(F) — Class A/B Coverage Test DPS Draw
if either of the Class A/B Coverage Tests is not satisfied on the related Determination Date, to make payments in accordance with the Debt Payment Sequence to the extent necessary to cause all Class A/B Coverage Tests to be satisfied on a pro forma basis;
§ 11.1.1.1(H) — Class C Coverage Test DPS Draw
if either of the Class C Coverage Tests is not satisfied, to make payments in accordance with the Debt Payment Sequence to the extent necessary to cause all Class C Coverage Tests to be satisfied;
§ 11.1.1.1(K) — Class D Coverage Test DPS Draw
if either of the Class D Coverage Tests is not satisfied, to make payments in accordance with the Debt Payment Sequence to the extent necessary to cause all Class D Coverage Tests to be satisfied;
§ 11.1.1.1(M) — Interest Diversion Test
during the Reinvestment Period, if the Interest Diversion Test is not satisfied, to deposit into the Collection Account as Principal Proceeds or to make payments in accordance with the Debt Payment Sequence;
§ 11.1.2 — Insufficiency of Funds (Operative Limit)
If on any Payment Date the amount available in the Payment Account is insufficient to make the full amount of the disbursements required by the Distribution Report, the Trustee shall make the disbursements called for in the order and according to the priority set forth under the Priority of Payments, subject to Section 13.1, to the extent funds are available therefor.
