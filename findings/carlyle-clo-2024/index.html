Carlyle Direct Lending CLO 2024-1 is a $431.6 million middle market CLO managed by Carlyle Global Credit Investment Management. Rated by S&P Global Ratings. Collateralised by senior secured loans.

Under simultaneous coverage test failures, available Interest Proceeds are insufficient to satisfy all Debt Payment Sequence draws.

Indenture dated October 29, 2024  ·  SEC EDGAR — unmodified filing  ·  Verified June 2026

Conditional
Stress Exposure Detected
Under simultaneous coverage test failures, the aggregate Debt Payment Sequence draws exceed available Interest Proceeds. Lower-priority obligations cannot be satisfied.
01 — The Finding

The Priority of Payments waterfall contains four independent coverage test triggers — clauses (F), (H), (K), and (M) of Section 11.1.1.1 — each directing payment through the Debt Payment Sequence when a test fails.

Each draw competes for the same pool of available Interest Proceeds. No provision caps the aggregate across simultaneous failures.

Section 11.1.2 instructs the Trustee to disburse in waterfall order to the extent funds are available. When multiple coverage tests fail at once, the combined Debt Payment Sequence obligation exceeds available proceeds. Lower-priority obligations go unmet.

02 — The Source Provisions
§ 11.1.1.1(F) — Class A/B Coverage Test DPS Draw
if either of the Class A/B Coverage Tests is not satisfied on the related Determination Date, to make payments in accordance with the Debt Payment Sequence to the extent necessary to cause all Class A/B Coverage Tests to be satisfied on a pro forma basis;
§ 11.1.1.1(H) — Class C Coverage Test DPS Draw
if either of the Class C Coverage Tests is not satisfied, to make payments in accordance with the Debt Payment Sequence to the extent necessary to cause all Class C Coverage Tests to be satisfied;
§ 11.1.1.1(K) — Class D Coverage Test DPS Draw
if either of the Class D Coverage Tests is not satisfied, to make payments in accordance with the Debt Payment Sequence to the extent necessary to cause all Class D Coverage Tests to be satisfied;
§ 11.1.1.1(M) — Interest Diversion Test
during the Reinvestment Period, if the Interest Diversion Test is not satisfied, to deposit into the Collection Account as Principal Proceeds or to make payments in accordance with the Debt Payment Sequence;
§ 11.1.2 — Insufficiency of Funds
If on any Payment Date the amount available in the Payment Account is insufficient to make the full amount of the disbursements required by the Distribution Report, the Trustee shall make the disbursements called for in the order and according to the priority set forth under the Priority of Payments, subject to Section 13.1, to the extent funds are available therefor.
